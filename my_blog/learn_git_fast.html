The importance of Version Control: How to learn Git fast

Version Control is essential to any programmer. According to Pro Git, written by Scott Chacon, Version control "is a system that records changes to a file or set of files over time so that you can recall specific versions later".


If I had to list the best resources for learning Version Control using Git, I would say they are (from beginners to advance):

  • GitHub tutorial
  • Pro Git book
  • StackOverFlow's answers on Git
